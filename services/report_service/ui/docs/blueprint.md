# **App Name**: Social Pulse

## Core Features:

- Topic Definition: Allows the user to define the analysis topic (brand,
  product, feature, etc.).
- Source Selection: Allows users to select which social media content sources
  to analyze (YouTube, Reddit, etc.).
- Analysis Type: Allows the analyst to specify whether to analyze overall
  topic sentiment or share of voice across different brands/products.
- Date Range Selection: Allows users to specify a start and end date for the
  analysis window.
- Create Report: Call the sentiment analysis pipeline to create a new report
  based on the specified parameters.
- List Reports: List all available reports.
- Report Details: Get details about a specific report, including a link to
  the report results.

- Primary color: Strong purple (#6200EE) to convey authority and insight for

- Background color: Light grey (#FAFAFA) to provide a clean, neutral backdrop
  for data visualization for Social Pulse.
- Accent color: Blue (#2962FF) to highlight key data points and interactive
  elements for Social Pulse.
- Headline Font: 'Space Grotesk' sans-serif for a modern, scientific look for
  Social Pulse.
- Body Font: 'Inter' sans-serif for readability and neutrality for Social
  Pulse.
- Use clear and informative icons to represent data sources, sentiment types,

- A clean and structured layout to present the complex data in an easily
